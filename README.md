# Aplicació Kanban de gestió de tasques

Aplicació web tipus Kanban per gestionar tasques, desenvolupada amb HTML, CSS i JavaScript.

## Funcionalitats
- Crear, editar i eliminar tasques
- Canviar estat (Per fer, En curs, Fet)
- Filtres per estat i prioritat
- Cerca per text
- Estadístiques de tasques
- Persistència amb localStorage
- Disseny responsive

## Guia ràpida d’ús
1. Omple el formulari i guarda una tasca
2. Mou-la entre columnes amb el selector d’estat
3. Utilitza els filtres i la cerca
4. Consulta les estadístiques
